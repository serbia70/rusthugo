## 创建 GitHub OAuth App

https://github.com/settings/applications/new：
- Application name: rusthugo-cms
- Homepage URL: https://rusthugo.vercel.app
- Authorization callback URL: https://rusthugo.vercel.app/api/callback

创建后会得到 Client ID 和 Client Secret。

## 方法二
1. 打开 https://github.com/settings/developers → New OAuth App
  - Application name: RustBlog Decap CMS
  - Homepage URL: https://rustblog-ivory.vercel.app
  - Authorization callback URL: https://rustblog-ivory.vercel.app/api/auth
2. 创建后复制 Client ID 和 Client Secret
3. 用以下命令设置到 Vercel：

npx vercel env add GITHUB_CLIENT_ID

npx vercel env add GITHUB_CLIENT_SECRET
