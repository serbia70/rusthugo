# docker相关命令

## 登陆

docker login ghcr.io -u serbia70

## 部署

docker compose down && docker compose pull && docker compose up -d

## 重启

docker compose restart


##看下docker 占了多少硬盘

docker system df

## 一键清理所有无用资源（停止的容器、未使用的镜像、网络、构建缓存）

docker system prune -a

## 如果有未使用的卷也清理

docker volume prune
