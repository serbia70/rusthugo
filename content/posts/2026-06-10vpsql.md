清理所有未使用的资源（包括停止的容器、未使用的网络和悬空镜像）:

```
docker system prune -f
```


查看 Docker 占用了多少空间
```
docker system df
```

查看当前目录下各文件夹的大小，并按大小排序
```
du -sh * | sort -rh
```

清理停止的容器
```
docker container prune -f
```

清理无用的构建缓存
```
docker builder prune -f
```
手动查看镜像
```
docker images
```

手动删掉那些你确定不再使用的陈旧版本镜像（假设镜像 ID 为 IMAGE_ID
```
docker rmi <IMAGE_ID>
```

深入 /var 目录，看看里面到底是谁在“吃”磁盘
```
sudo du -sh /var/* 2>/dev/null | sort -rh
```

查询/var/lib
```
sudo du -sh /var/lib/* 2>/dev/null | sort -rh | head -n 10
```


安全地清理 containerd 缓存

```
systemctl restart docker
```

清理 Docker 没用的 Volume
```
docker volume prune -f
```


定期清理不再使用的容器和数据卷,每隔几周运行一次
```
docker container prune -f
docker volume prune -f
```
