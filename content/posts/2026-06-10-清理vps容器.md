清理所有未使用的资源（包括停止的容器、未使用的网络和悬空镜像）：

```
docker system prune -f
```


查看 Docker 占用了多少空间
```
docker system df
```

查看当前目录下各文件夹的大小，并按大小排序
```
du -sh * | sort -rh
```

清理停止的容器
```
docker container prune -f
```

清理无用的构建缓存
```
docker builder prune -f
```
手动查看镜像
```
docker images
```

手动删掉那些你确定不再使用的陈旧版本镜像（假设镜像 ID 为 IMAGE_ID
```
docker rmi <IMAGE_ID>
```
