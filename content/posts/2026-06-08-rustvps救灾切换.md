切换n1
```
VPS_IP=vps的ip bash n1.sh
```

切换vps
```

VPS_IP=vps的ip bash vps.sh https://rustvps.serbia70.com
```

```
chmod +x /opt/rustvps/food-server
```

```
systemctl stop food-server-backup
```

```
systemctl restart food-server-backup
```
查看打印mqtt

```
grep "mqtt_broker\|mqtt_port" /opt/rustvps/config.json
```

检查 usblp 模块是否已加载
```
lsmod | grep usblp
```

查看打印
```
journalctl -u mqtt-printer -f
```

查看N1后端状态
```
journalctl -u food-server-backup -f
```

查看mqtt状态
```
journalctl -u mosquitto -f
```
