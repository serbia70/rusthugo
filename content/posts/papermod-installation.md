<!-- more -->

## 安装主题

```bash
git clone https://github.com/serbia70/rusthugo
cd rusthugo
zola serve
```

## 配置说明

编辑 `config.toml` 文件，配置站点基本信息、导航菜单、社交图标等。

## 自定义

通过 `override.css` 和自定义模板扩展主题功能。
